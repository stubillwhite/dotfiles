# cheatsheet #

## Basics ##

- `Ctrl+\` - Leader
- `:` - Command
- `d` - Disconnect

## Config reloading ##

- `I` - Install plugins
- `r` - Reload config

## Panes ##

- `|` - Split pane vertically
- `-` - Split pane horizontally
- `x` - Kill current pane

Alt and arrow keys, or prefix and Vim keys for navigating panes

## Windows ##

- `w` - Select window
- `n` - Next window
- `p` - Previous window
- `!` - Break pane into new window
- `&` - Kill current window
- `,` - Name current window
- `.` - Move current window
