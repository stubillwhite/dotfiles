# cheatsheet #

## Basics ##

- `Ctrl+|` - Leader
- `:` - Command
- `d` - Disconnect

## Panes ##

- `|` - Split pane vertically
- `-` - Split pane horizontally
- `x` - Kill current pane

## Windows ##

- `w` - Select window
- `n` - Next window
- `p` - Previous window
- `!` - Break pane into new window
- `&` - Kill current window
- `,` - Name current window
- `.` - Move current window
